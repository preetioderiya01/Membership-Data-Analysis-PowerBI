# Membership Dashboard - Power BI

## Project Overview
This project is an interactive Business Intelligence dashboard developed using Power BI to analyze membership data across different regions, membership types, and payable amounts.

The dashboard helps in understanding membership distribution, identifying top regions, and analyzing revenue trends over time.

## Tools Used
Power BI  
Excel 
Power Query
Data Modeling  
DAX  
Data Visualization

## Key Features
Region-wise membership analysis  
Top 5 regions with highest membership  
Top 5 regions with lowest membership  
Year-wise payable trend analysis  
Membership type payable comparison  
Customer-level membership details

## Dashboard Pages

### 1. Membership Dashboard
Shows region-wise membership distribution and payable trends.

### 2. Membership Dashboard
Dispaly overview of membership distribution, member types, and customer payable trends across different years.

### 3. Membership Analysis
Displays customer information including member type, membership type, and payable details.

### 4. Member Analysis
Detailed member information with payable amount by year.

### 5. Advanced Analysis
Provides deeper insights into regional payable trends and customer data.

## Key Insights
HO-MUM region has the highest membership count.
Payable amounts show consistent growth from 2019 to 2023.
ZASSOC membership type contributes the highest payable value.

## Dataset
The dataset contains the following fields:

Customer Name  
Region  
Member Type  
Membership Type  
Type of Firm  
Payable Amount  
Year and Month
